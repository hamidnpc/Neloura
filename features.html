What you can do with Neloura?

Neloura is a comprehensive platform for astronomical data analysis, specializing in FITS image processing, catalog management, and multi-wavelength visualization. Built on a Python FastAPI backend with a sophisticated JavaScript frontend, and powered by the Astropy ecosystem, Neloura provides an intuitive web-based interface for exploring astronomical datasets with different analysis capabilities.

Neloura

Powered by Python and JavaScript, Neloura delivers in-browser visualization of large astronomical mosaics, catalogs, SEDs, and interactive source finding—no installation required. It is designed for the massive datasets expected from SKA, Roman, and other next-generation missions.
